# mostly-harmless-api-sample
Harmless, Mostly.

## installation
1. git clone
    ```
    git clone git@github.com:quag-cactus/mostly-harmless-api-sample.git
    ```

1. Install dependencies using pdm
    ```
    pdm install
    ```

1. Install git hooks using pre-commit
    ```
    pdm run pre-commit install
    ```
